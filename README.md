# Enterprise Geo Metadata Discovery Reprojector

A Python-based workflow to:

- discover enterprise spatial metadata
- identify duplicates and archive candidates for data-cleaning plans
- perform coordinate reprojection (for example, GDA94 to GDA2020)
- detect where files are used in user workflows
