# Enterprise_Geo_Metadata_Discovery_Reprojector-
A python based workflow to discover enterprise spatial metadata, plan cleaning of duplicate and archive data and perform reprojection (such as GDA 94 to 2020). It also discovers file use in user workflows and corporate systems and either fixes them automatically or provides instructions to users on how to update there systems.
